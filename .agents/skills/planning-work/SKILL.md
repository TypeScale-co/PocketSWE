# Planning Work

Use this skill before starting implementation of any feature or epic. It governs the
path from a raw feature request to a closed Epic: discovery, architecture, decomposition,
dependency graphing, execution, integration, and close.

Read [docs/work-protocol.md](../../../docs/work-protocol.md) and follow it directly. It
is the single source of truth for the seven-step protocol
(Discover → Architect → Decompose → Build Dependency Graph → Execute →
Integrate & Review → Close), including the required North Star content and the
completion conditions for each step.

Where the protocol references the Architecture Manifest, use
[docs/architecture.md](../../../docs/architecture.md).
